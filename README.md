# 🚗 DriveFlow - Conectando Alunos e Instrutores de Trânsito

O **DriveFlow** é uma plataforma web desenvolvida para conectar alunos que desejam obter a CNH ou aprimorar suas habilidades de direção com instrutores autônomos credenciados pelo DETRAN. O sistema resolve problemas de agendamento, transparência no desempenho do aluno e verificação de credenciais de instrutores.

## 🚀 Funcionalidades Principais

- **Gestão de Usuários:** Cadastro de Alunos e Instrutores com perfis distintos.
- **Verificação de Credenciais:** Sistema de envio de documentos para instrutores.
- **Agendamento Inteligente:** Alunos podem marcar aulas conforme a disponibilidade do instrutor.
- **Histórico de Desempenho:** Registro técnico detalhado de cada aula realizada.
- **Avaliações e Feedback:** Sistema de notas (1 a 5 estrelas) para garantir a qualidade do serviço.

## 🛠️ Tecnologias Utilizadas

- **Spring Data JPA** 
- **PostgreSQL** 
- **Docker & Docker Compose**

### Frontend
- **Angular 17+**
